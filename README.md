# Alpha - Business AI Controller

Alpha is an advanced AI agent control dashboard designed for premium business communication management.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Database:** Supabase
- **AI Core:** Gemini 3 Flash
- **Orchestration:** n8n
- **UI:** Tailwind CSS with Glassmorphism

## Features
- Real-time conversation monitoring
- Manual intervention (Panic Button)
- Lead analysis & scoring
- Agent personality configuration

## Getting Started
1. Clone the repo
2. Create `.env.local` with Supabase credentials
3. Run `npm install`
4. Run `npm run dev`
